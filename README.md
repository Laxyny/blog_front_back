# blog_Theo_Kevin
 
Pour lancer le projet, il suffit de lancer la commande suivante à la racine du projet :


- Pour le back-end :

```bash
npm run dev
```

- Pour le front-end :

```bash
ng serve
```


Le projet est accessible à l'adresse suivante :

```bash
http://localhost:4200/
```
